# 요구사항 정의서

## 프로젝트 목표

청년들이 정부 혜택, 지원 사업, 공모전 등의 정보를 찾기 위해 여러 사이트를 방문하고 복잡한 자격 조건을 직접 확인해야 하는 문제를 해결한다.

사용자가 자신의 정보를 한 번 입력하면 받을 수 있는 혜택을 자동 판별하고, 신청 기간을 관리할 수 있는 플랫폼을 구축한다.

---

# 사용자 기능

## 회원가입 및 로그인

### 기능 설명
사용자가 서비스를 이용하기 위해 회원가입 및 로그인을 수행한다.

### 요구사항
- 회원가입
- 로그인
- 로그아웃
- 세션 기반 인증

---

## 사용자 정보 관리

### 기능 설명
정책 자격 판별에 필요한 개인정보를 입력 및 수정한다.

### 입력 정보

- 나이
- 거주 지역
- 가구원 수
- 월소득
- 연소득
- 취업 여부
- 학생 여부
- 무주택 여부
- 중소기업 재직 여부
- 기초생활수급자 여부
- 차상위 여부
- 장애 여부

---

## 중위소득 자동 계산

### 기능 설명

사용자가 입력한 소득과 가구원 수를 바탕으로 중위소득 비율을 자동 계산한다.

### 결과 예시

- 중위소득 82%
- 중위소득 54%

---

## 정책 자격 판별 (Eligibility Engine)

### 기능 설명

사용자 정보와 정책 조건을 비교하여 신청 가능 여부를 자동 판별한다.

### 판별 결과

- 신청 가능
- 신청 불가능
- 추가 정보 필요

### 결과 예시

신청 가능

- 나이 조건 충족
- 지역 조건 충족
- 무주택 조건 충족

신청 불가능

- 중위소득 기준 초과

---

### 기능 설명

등록된 정책을 조회한다.

### 제공 기능

- 카테고리별 조회
- 개인 맞춤 정책 조회

---

## 정책 상세 조회

### 제공 정보

- 정책 설명
- 지원 내용

- 신청 링크
